# 第10章 文件和异常



## 概览

​	学习处理文件和保存数据可让你的程序使用起来更容易。



## 10.1 从文件中读取数据

​	文本文件可存储的数据量多的难以置信。



### 10.1.1 读取整个文件

​	pi_digits.txt

```
3.122222
	3333
	1111
```

​	file_reader.py

```python
with open('pi_digits.txt') as file_object:
	contents = file_object.read()
	print(contents)
```

​	输出结果

```
3.122222
        3333
        1111
```

​	相比于原始文件，该输出唯一不同的地方是末尾多了一个空行。为何多出现这个空行？因为read()到达文件末尾返回一个空字符串，而这个空字符串显示出来时就是一个空行。要删除多余的空行，添加rstrip()

```python
with open('pi_digits.txt') as file_object:
	contents = file_object.read()
	print(contents.rstrip())

```

​	输出结果

```
3.122222
        3333
        1111
```



### 10.1.2 文件路径

​	Python将在当前执行的文件所在目录中查找文件。

​	根据组织文件的方式,有时可能要打开不在程序文件中所属目录的文件。

​	Linux和OS X中

```
with open('text_files/pi_digits.txt') as file_object:
```

​	在windows中

```
with open('text_files\pi_digits.txt') as file_object:
```

​	还可以绝对路径存储到一个变量中。

```
file_path = 'd:\data\python_work\pi_digits.txt'
with open(file_path) as file_object:
	contents = file_object.read()
	print(contents.rstrip())
```



### 10.1.3 逐行读取

​	读取文件时,常常需要检查其中的每一行

```python
file_path = 'd:\data\python_work\pi_digits.txt'
with open(file_path) as file_object:
	for line in file_object:
		print(line)

```

```
3.122222

        3333

        1111

```

​	打印每一行时，发现空白行更多了

```python
file_path = 'd:\data\python_work\pi_digits.txt'
with open(file_path) as file_object:
	for line in file_object:
		print(line.rstrip())
```

```
3.122222
        3333
        1111
```



### 10.1.4 创建一个包含文件各行内容的列表

​	使用关键字with时,open()返回的文件对象只有with代码块可用。

​	如果要在with代码块外访问文件的内容，可在with代码块内将文件的各行存储在一个列表中，并在with代码块外使用改列表

```python
file_path = 'd:\data\python_work\pi_digits.txt'
with open(file_path) as file_object:
	lines = file_object.readlines()

for line in lines:
	print(line.rstrip())

```

```
3.122222
        3333
        1111
```



### 10.1.5 使用文件的内容

```python
filename = 'pi_digits.txt'
with open(filename) as file_object:
	lines = file_object.readlines()

```
