- 注释分析表

  主要为其设置了如下几类标志

  1、当注释中注释是否，或者出现1--，0--后选择判断表示代码，数据元选择判断标识代码

  2、当注释中出现code,dm,‘代码',等关键字段后，标注为代码，并默认与代码匹配

  3、当注释中出现`0: 1: 2: 3:`等疑似代码，需要为其标注代码

  4、当注释中出现非汉字外的其他字段

```
create table sjzl.ysys_sjzl_annotate_analyze(annotate varchar(1000),frenquency int4,annotate_flag int4,annotate_analyze varchar(1000),standar_source varchar(100),standar_cn_name varchar(100),limit_en_name varchar(100),limit_cn_ename varchar(100));
comment on table sjzl.ysys_sjzl_annotate_analyze is '注释分析'；
```
