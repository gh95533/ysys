## postgresql 是什么

​	postgresql 数据库是目前功能最强大的开源数据库，支持丰富的数据类型(JSON,JSONB,数组类型)和自定义类型。



### 发展历时 

​	前身是ingres

​	postgres95开源

​	postgres9.x进入黄金时代



### postgresql 发展趋势

​	开源数据库

​	稳定可靠

​	开源省钱

​	支持广泛

​	社区活跃



## 和其他数据库对比



### postgresql vs mysql



​	mysql 缺点

​	功能不够强大，sql多表关联和子查询

​	性能优化工具和度量信息不足

​	在线操作功能较弱

​	postgresql优点

​	支持各种查询

​	性能优化工具和度量信息丰富

​	在线操作功能号



### postgresql vs oracle

​	oracle牛

​	postgresql 暂时不足
