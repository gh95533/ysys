## operation

```
select gp_segment_id,count(1) from table group by gp_segment_id;
```

​	其中gp_segment_id是表table的字段,pg_attribute中就有该字段,是gp创建表时默认自带(oracle rowid),gp_segment_id对着是`gp_segment_configuration的content`

​	这个sql只能查看某张表，只是大致上查看了数据情况

​	另外一个方案是使用系统表，通过系统表来帮助研判情况，那么就可以查询出来多张表的状态了。

```
select bdirelid::regclass,* from gp_toolkit.gp_bloat_diag ;
```

​		其中bdirelid::regclass就是表名

​	还有一张表

```
SELECT btdrelid::regclass,* FROM gp_toolkit.gp_bloat_expected_pages where btdrelpages>btdexppages
```

​	btdrelid::regclass就是表名

​	btdrelpages:实际的pages页面

​	btdexppages:期待的pages页面

## continue

​	确定了膨胀表,分布异常表，就要重建表

```
begin
lock bbb;
create table aaa (..) distributed by (..)
insert into aaa select * from bbb;
drop table bbb;
alter table aaa rename to bbb;
end;
```
