##  我的第一个JAVA程序

```
public class HelloWorld{
    public static void main(String[] args){
        System.out.println("Hello World");
    }
}
```

*注:String[] args与String args[]都是可以执行，但推荐使用String[] args,这样就可以避免歧义和误读*

将内容拷贝到HelloWorld.java,后通过`javac HelloWorld.java`编译出来一个文件`HelloWorld.class`后，执行命令`java HelloWorld`出现结果`Hello World`

- JavaEE(J2EE)        (Java 2 Platform,Enterprise Edition，java平台企业版)
