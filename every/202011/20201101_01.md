## Operation

### pip 导出本地已有的依赖包

```
pip freeze > requirements.txt
```

 requirements.txt 文件内容,每个依赖包的版本

```
certifi==2020.6.20
chardet==3.0.4
id-validator==1.0.12
idna==2.10
numpy==1.19.2
pandas==1.1.2
psycopg2==2.8.4
pyodps==0.10.0
python-dateutil==2.8.1
pytz==2020.1
requests==2.24.0
six==1.15.0
urllib3==1.25.10
xlrd==1.2.0
xlutils==2.0.0
xlwt==1.3.0
```

### 将依赖包下载到本地

```
pip download -r requirements.txt -d . -i http://mirrors.aliyun.com/pypi/simple/ --trusted-host mirrors.aliyun.com
```

### 将文件和依赖包拷贝另外一台服务器

...

### 安装本地依赖包

```
pip install --no-index --find-links=. -r requirements.txt
```
